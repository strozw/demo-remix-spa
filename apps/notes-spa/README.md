# Remix SPA Monorepo Demo

## Setup

```sh
pnpm install
```

## Develop

### Develop Apps & Packages

```sh
pnpm dev
```

### Build Apps & Packages

```sh
pnpm buld
```

### Develop Apps & Packages Components in Storybook

```sh
pnpm storybook
```

### Build Apps & Packages Storybook

```sh
pnpm build-storybook
```

### Tests Apps & Packages

```
pnpm test
```
